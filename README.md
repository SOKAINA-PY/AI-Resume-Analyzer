# 📄 AI Resume Analyzer

An intelligent web application that analyzes resumes, detects technical skills, calculates an ATS score, and compares resumes with job descriptions.



## 🚀 Features

- 📄 Upload PDF Resume
- 📝 Extract Resume Text
- 🧠 Detect Technical Skills
- 📊 ATS Score Calculation
- 📈 Progress Bar
- ✅ Matched Skills
- ❌ Missing Skills

- PDF Resume Upload
- PDF Job Description Upload
- Resume Text Extraction
- Skills Detection
- ATS Score Calculation
- Similarity Analysis
- Matched Skills
- Missing Skills
- AI Recommendations
